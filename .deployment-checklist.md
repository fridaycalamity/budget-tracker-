# Pre-Deployment Checklist

Use this checklist before deploying to production.

## Code Quality

- [ ] All tests pass (`npm run test`)
- [ ] No linting errors (`npm run lint`)
- [ ] Code is properly formatted (`npm run format:check`)
- [ ] No TypeScript errors (`tsc -b`)
- [ ] No console errors in browser

## Build Verification

- [ ] Production build succeeds (`npm run build`)
- [ ] Build output is in `dist/` directory
- [ ] Bundle size is reasonable (< 500KB gzipped)
- [ ] Preview build works locally (`npm run preview`)

## Functionality Testing

- [ ] All pages load correctly
- [ ] Navigation works (Dashboard, Transactions, Budget Goals, Settings)
- [ ] Theme toggle works and persists
- [ ] Add transaction works with validation
- [ ] Delete transaction works with confirmation
- [ ] Filters work (type, category, date range)
- [ ] Sorting works (date, amount, asc/desc)
- [ ] Charts display correctly
- [ ] Budget goals work
- [ ] Clear all data works
- [ ] localStorage persistence works

## Responsive Design

- [ ] Mobile view (320px - 767px) works
- [ ] Tablet view (768px - 1023px) works
- [ ] Desktop view (1024px+) works
- [ ] Touch interactions work on mobile
- [ ] Hamburger menu works on mobile

## Accessibility

- [ ] Keyboard navigation works
- [ ] Focus indicators are visible
- [ ] ARIA labels are present
- [ ] Color contrast is sufficient
- [ ] Screen reader compatible

## Performance

- [ ] Initial load time < 3 seconds
- [ ] No unnecessary re-renders
- [ ] Images are optimized
- [ ] Assets are cached properly

## Deployment Configuration

- [ ] `vercel.json` is present (for Vercel)
- [ ] `netlify.toml` is present (for Netlify)
- [ ] `vite.config.ts` has production optimizations
- [ ] Base URL is configured correctly (if needed)

## Documentation

- [ ] README.md is up to date
- [ ] DEPLOYMENT.md is complete
- [ ] All spec files are current

## Post-Deployment

- [ ] Verify live site loads correctly
- [ ] Test all functionality on live site
- [ ] Check browser console for errors
- [ ] Test on multiple browsers (Chrome, Firefox, Safari, Edge)
- [ ] Test on mobile devices
- [ ] Verify analytics/monitoring (if configured)
